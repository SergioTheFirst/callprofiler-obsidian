# Business Context

## What this system does
Local multi-user system for processing phone call recordings.
Chain: phone records → PC processes → user gets Telegram digest + Android overlay on incoming call.

## Who uses it
- Serhio (owner, main user)
- Potentially 1-2 more users on same machine

## Core value
- Know who is calling BEFORE you answer (overlay)
- Get summary of every call automatically
- Search all call history by text
- Track promises and action items from calls

## Scale
- ~18,000 existing transcripts loaded
- Hundreds of new calls monthly
- Single Windows PC, local only — no cloud

## Success criteria
- File → Telegram digest in under 5 minutes
- Android overlay shows context on incoming call
- System runs a week without manual intervention
